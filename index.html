GeminiFeedbackCAD

A FreeCAD macro that uses Google's Gemini API to generate 3D models with human feedback

Download

Features

  • Generate 3D models from text descriptions
  • Refine models with human feedback
  • Easy installation with automatic setup
  • Works with FreeCAD 1.0.0+

Installation

  1. Download the ZIP package
  2. Extract the files
  3. Run install.py
  4. Open FreeCAD and find the macro in Macro → Macros...
